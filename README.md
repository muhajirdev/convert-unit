# Convert Unit

Convert values to the closest unit

## Usage

```bash
npm install --save convert-unit
```

```javascript
const { toClosestUnit } = require("convert-unit");

console.log(toClosestUnit(10000000));
// { value: 10, size: 'MB' }
```
